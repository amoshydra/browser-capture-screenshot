Browser Capture Screen utilise browser's
[`getDisplayMedia`](https://developer.mozilla.org/en-US/docs/Web/API/MediaDevices/getDisplayMedia)
API to capture screenshot from the page.


## Getting started

```bash
npm install browser-capture-screen
```

import {} from "browser-capture-screenshot";
